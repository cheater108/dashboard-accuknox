# React + TypeScript + Vite

## Tech Stack

**Client:** React, Tailwind

**Build Tool:** Vite

## Run Locally

```bash
  git clone https://github.com/cheater108/dashboard-accuknox
```

Go to the project directory

```bash
  cd dashboard-accuknox
```

Install dependencies

```bash
  npm install
```

Run the dev server

```bash
  npm run dev
```
